# PDF 저장 가이드

## 📄 권장: 텍스트 레이어가 있는 PDF

### 알PDF로 저장하는 방법
1. 알PDF 실행
2. 파일 → 다른 이름으로 저장
3. **"텍스트 레이어 유지"** 또는 **"검색 가능한 PDF"** 옵션 선택
4. 저장

### 다른 프로그램에서 저장
- **Microsoft Word/Excel**: 파일 → 내보내기 → PDF/XPS → 옵션에서 "문서 구조 태그" 체크
- **Adobe Acrobat**: 파일 → 다른 이름으로 저장 → 검색 가능한 PDF
- **Chrome/Edge**: 인쇄 → PDF로 저장 (텍스트 유지됨)

## 🖼️ 대안: 이미지 기반 PDF

이미지 PDF를 사용하는 경우:
1. **엑셀 파일을 먼저 로드** 필수
2. PDF 페이지 순서와 엑셀 송장번호 순서가 일치해야 함
3. 프로그램이 자동으로 매핑

## ✅ 확인 방법

PDF 파일을 열어서:
- 텍스트를 선택할 수 있으면 ✅ 텍스트 레이어 있음
- 텍스트를 선택할 수 없으면 ❌ 이미지 PDF

## 💡 팁

- 가능하면 항상 텍스트 레이어가 있는 PDF 사용
- 이미지 PDF는 엑셀 파일과 함께 사용해야 함
